# Flutter Calculator App

A basic calculator application built with Flutter that allows users to perform simple arithmetic operations such as addition, subtraction, multiplication, and division.

##  Features

* Perform basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
* User input using TextFields
* Handles invalid input gracefully
* Clean and simple UI


## Tech Stack

* Flutter
* Dart

### Prerequisites

* Flutter SDK installed
* Android Studio / VS Code
* Emulator or physical device

## How It Works

* Uses a `StatefulWidget` to manage user inputs and result.
* `TextEditingController` captures numeric input.
* Arithmetic operations are handled via a `switch` statement.
* Division by zero is safely handled by returning 0.
